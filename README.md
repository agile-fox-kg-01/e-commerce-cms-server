## API Documentation

----
  **Login User**
----
  login existed user

* **URL**

  /login

* **Method:**

  `POST`

* **Request Headers**

  None

* **URL Params**

  None

* **Data Params**

  | key | value | required |
  | :---: | :---: | :---: |
  | email | <user_email> | true |
  | password | <user_password> | true |

* **Success Response:**

  * **Code:** 200 OK <br />
    **Content:**
    ```json
    {
        "token": "user-token",
        "email": "user-email",
        "role": "user-role"
    }
    ```

* **Error Response:**
    * **Code:** 400 Bad Request <br />
        **Content:**
        ```json
        { "errors" : "invalid username/password" }
        ```
    
    * **Code:** 500 Internal Server Error <br />
        **Content:** 
        ```json
        { "error" : "Internal Server Error" }
        ```
